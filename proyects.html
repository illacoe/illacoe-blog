Clone Apple Music

Este proyecto fue realizado con la finalidad de aprender a crear un servidor con Express y ademas implementar una aplicación con ReactJS. El primer procedimiento que realice fue comenzar con la plantilla del proyecto, una vez finalizada procedi a tranformarlo a una aplicacion usando la libreria de facebook, y finalmente creamos el servidor con la libreria de nodejs. para crear el servidor y unirlo a una API, tuve algunos problemas. leyendo la documentacion de Express llegue a la conclusion de que necesitaba usar web sockets. Finalmente pude terminar el mini proyecto.
